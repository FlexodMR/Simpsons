NAME

Tk::After - support class for Tk::Widget::after

SYNOPSIS 

  $id = $widget->after(time,callback);
  $id = $widget->afterIdle(callback);
  $widget->afterCancel($id);


  $id = $widget->repeat(period,callback);

Internally this class is used to implement above. 

  $id = Tk::After->new($widget,$time,'method',callback);
  $id->cancel;

DESCRIPTION

This class is a wrapper used by Tk::Widget::after to auto-cancel after calls when a widget is destroyed.

callback is a normal perl/Tk callback. Method is either 'once' or 'repeat'

This is first attempt at the code and interface is likely to change. So for the time being at least use $widget->after(...) interface.

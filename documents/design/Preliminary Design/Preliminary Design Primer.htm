Preliminary Design Primer

 

This document is a first-draft preliminary design of the core gameplay features. It is not representative of the final preliminary design to be delivered on March 15th 2002. Specifically:

 

  • Not yet scoped down by tech or art leads.
  • Some art sections not available (animation, vehicle lists, etc.)
  • Several design sections not yet available (controls, presentation, sound, stories, etc.)

 

The documents in this section are not yet organized. Here’s the recommended reading order to get a feel for the game:

 

  1. High Concept (one folder up). Read this first.
  2. Vehicle Chase AI. This is the core of the game’s fun.
  3. Vehicle Escape AI. This is the other core gameplay component.
  4. Switching Vehicles In Game. The on-foot component relating to gameplay.
  5. Collectibles. Not truly gameplay, these are the reasons we’ll give the user to run around the world.
  6. Multiplayer. The 2-player game mode.
  7. Player Navigation System. How the user is directed through the game world.
  8. Mission Flow. How the user moves from objective to objective.
